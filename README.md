## 🧩 웹 서비스 소개

### **여행에 조각을 더하다, NFT Sticker + _READ, WRITE, SELL_** ✈️

> 대한민국 방방곡곡을 돌아다녀보세요.🚶🏻‍♀️🚶🏻
>
> 특별한 지역의 **QR을 읽고** 특별한 **NFT 스티커**를 발급받고, 나만의 **여행 다이어리를 쓸** 수 있어요.
>
> 사용하지 않는 NFT는 마켓에서 **팔아**보세요

## 💻 실행 방법

### Client 실행

1. **원격 저장소 복제**

```bash
$ git clone https://lab.ssafy.com/s07-blockchain-nft-sub2/S07P22A607.git
```

2. **프로젝트 폴더로 이동**

```bash
$ cd frontend
```

3. **필요한 node_modules 설치**

```bash
$ yarn install
```

4. **개발 서버 실행**

```bash
$ yarn start
```
